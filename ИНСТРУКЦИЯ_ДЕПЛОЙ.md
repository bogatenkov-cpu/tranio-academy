Все команды нужно выполнять из директории проекта:


## Шаг 1: Проверка репозитория

```bash

cd /Users/evgeniibogatenkov/Desktop/tranio-academy

git status
```

Если видите ошибку "not a git repository", значит вы не в той директории.

```bash

git add .

# Проверьте, что добавилось
git status
```

## Шаг 3: Создайте коммит

```bash
git commit -m "Ready for deployment: added all questions and fixes"
```

---

## Шаг 4: Подключите удаленный репозиторий (если еще не подключен)

### Вариант A: У вас уже есть репозиторий на GitHub

```bash
# Проверьте, есть ли уже remote
git remote -v

# Если нет, добавьте (замените YOUR_USERNAME на ваш GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/tranio-academy.git

# Или если используете SSH
git remote add origin git@github.com:YOUR_USERNAME/tranio-academy.git
```

### Вариант B: Создайте новый репозиторий на GitHub

1. Зайдите на https://github.com
2. Нажмите "New repository"
3. Название: `tranio-academy`
4. **НЕ** добавляйте README, .gitignore или лицензию (они уже есть)
5. Нажмите "Create repository"
6. Скопируйте URL репозитория
7. Выполните:

```bash
git remote add origin https://github.com/YOUR_USERNAME/tranio-academy.git
```

---

## Шаг 5: Отправьте код на GitHub

```bash
# Отправьте код
git push -u origin main
```

Если получите ошибку, попробуйте:
```bash
git push -u origin main --force
```
(⚠️ Используйте `--force` только если уверены!)

---

## Шаг 6: Деплой на Vercel

1. Зайдите на https://vercel.com
2. Войдите через GitHub
3. Нажмите "Add New Project"
4. Выберите репозиторий `tranio-academy`
5. Настройки (обычно автоматически определяются):
   - **Framework Preset:** Next.js
   - **Root Directory:** `./`
   - **Build Command:** `npm run build`
   - **Output Directory:** `.next`
6. Нажмите "Deploy"
7. Дождитесь завершения (2-3 минуты)
8. Получите домен: `tranio-academy.vercel.app`

---

## Шаг 7: Подключение своего домена (опционально)

1. В настройках проекта Vercel → "Domains"
2. Добавьте ваш домен (например, `academy.tranio.ru`)
3. Следуйте инструкциям по настройке DNS:
   - Добавьте CNAME запись в DNS вашего домена
   - Укажите значение, которое даст Vercel

## Если что-то пошло не так:

### Проблема: "fatal: not a git repository"
**Решение:** Убедитесь, что вы в правильной директории:
```bash
cd /Users/evgeniibogatenkov/Desktop/tranio-academy
pwd  # Должно показать: /Users/evgeniibogatenkov/Desktop/tranio-academy
```

### Проблема: "remote origin already exists"
**Решение:** Проверьте существующий remote:
```bash
git remote -v
# Если нужно изменить:
git remote set-url origin https://github.com/YOUR_USERNAME/tranio-academy.git
```

### Проблема: Ошибки при push
**Решение:** Проверьте права доступа к репозиторию на GitHub
